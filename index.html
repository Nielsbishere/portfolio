Wisp

Real-time raytracing


Description

Wisp was a real-time raytracing library that could've been integrated into your favorite engine or tool, made by a team of nine students. It was used in a Maya extension that allowed you to view a deferred, full raytracing, hybrid and path traced preview of the current scene.
The goal of the project was to allow artists to have a quick overview of what their end-product would look like; saving time and improving their workflow. Maya didn't provide that in the regular viewport, so the deferred preview was a step up to view the effects of modeling real-time. Previewing the reflections, global illumination and shadows is a nice bonus for RTX and GTX Pascal cards.

OxC3

Cross-platform framework

Description

Oxsomi Core 3 is a C framework that abstracts graphics and platform APIs, with Vulkan and DirectX12 backends with raytracing support, a shader pipeline targeting DXIL and SPIR‑V, and runs across Windows, Linux, macOS, and Android.
Built entirely from scratch, it includes systems for memory management, math/vector types, platform abstraction, file I/O, encryption, hashing, and custom file formats.
Core 3 evolved from Core 1, which had a Vulkan-only renderer and supported Android and Windows.

Harpoon

Unity C# Injector

Description

Harpoon was a project that was used in Shiploader; a modloader for the game Raft. This API was intended to be used by the modding community as an alternative. This was due to the existing modding tool using the source code of the game and distributing it and mods being very version dependent.
The alternative was code injection and not just patching a dll; where an executable requests Unity to load mods, which use a library for wrapping the low-level code.
This is accomplished by loading a C DLL into memory, which calls functions provided onto a heap; these functions are Unity mono functions, which load a C# dll and execute it. The C# dll is then able to access everything in Unity through C# reflection and Harmony through patching functions (More info here).

NDS Tooling

NDS Rom editor & Armulator

Description

A combined suite of tools for reverse engineering NDS games.
NFSU is a modular ROM editor that supports viewing and editing palettes, tilemaps, archives, and other formats common in DS games.

Mineblowers

Custom controller game

Description

Mineblowers was a marmalade game made in 3 weeks, which utilises a custom controller. The idea was to create a unique user experience with an unexpected controller.
We chose for a bike pump as a controller, because it was easy to obtain and it fit in with our minecart-style game.
The faster you pump, the faster you travel. If you travel too fast, you fall off and have to start over. The fastest score wins, this was implemented with a simple leaderboard. Since one of my team members had worked on this at Rail Recon before, I spoke to him and implemented it with a tool in our project.
